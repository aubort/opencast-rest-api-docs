      Method / Path:
      GET /{eventId}/messages
    

    

      Description:
      Returns the event messages as JSON
    

    

      Path params:
      
        eventId: The event identifier
      
    

    

      Optional (query) params:
      
        sort: The sort order.  May include any of the following: DATE OR SENDER.  Add '_DESC' to reverse the sort order (e.g. DATE_DESC).
      
    

    

      Response formats:
      
        application/json 
      
    

    

      Status codes:
      
        200: OK, The event messages as JSON. 

        404: Not Found, The event has not been found 

        400: Bad Request, Invalid SORT type, it was not DATE, DATE_DESC SENDER or SENDER_DESC 

        401: Unauthorized, If the current user is not authorized to perform this action
      		
    

    

      Notes:
      
        
    
          
  • Return value description: Returns the event messages as JSON
    • 
        

      		
    

    

      Testing:
